\conclusion

В ходе выполнения данной работы:

\begin{itemize}
\item Разработан и реализован на языке С алгоритм расчёта термодинамических функций по модели Томаса-Ферми с поправками с заданной точностью.

\item Вычислены теплоёмкость и давление на изохорах для алюминия.

\item Построены границы применимости модели Томаса-Ферми по отношению к квантовой и обменной поправкам для водорода и алюминия по энергии при относительных поправках $ 0.1, 0.5, 1.0 $.

\item Те же границы построены для тепловой части. Отмечено, что область применимости в этом случае действительно расширилась в область низких температур.

\item В дальнейшем планируется разработать алгоритм расчёта оболочечной поправки.

\end{itemize}
